Programa hecho en Tkinter para aprender idiomas.

Está desarrollado con pandas (para procesar las palabras) y Tkinter (Para la interfaz gráfica)

El programa lee un archivo (llamado french_words.csv) el cual lee las palabras y las inserta en las tarjetas. También
cuando sale de la aplicación genera un archivo (words_to_learn.csv) que guarda en un fichero csv las palabras que no se recuerdan.
